安装本软件的最低配置要求

如果要安装本软件，必须满足以下最低的配置要求。

系统版本 Windows Vista
Windows XP
Windows 7 (32 位，64 位)
Windows 8 (32 位，64 位)
Windows 10 (32 位，64 位)
CPU 至少 800MHz
内存 操作系统内存至少 512MB
显卡内存至少 32MB
可用硬盘容量 至少 200MB
串口通信功能 COM端口 (通信速度 2400bps~~115200bps)

注意事项
软件已经过大量测试，能够在上述的系统安装使用成功，已经在本公司的技术、生产、调试部门成功安装，并且已对发现的问题打上相应补丁，基本不存在安装失败的事情，对操作系统比较古老的版本，如果安装完后运行失败，请及时联系我们，我们将会在第一时间解决。
